# Kata Suite (20)

Goal: teach Rust with real compiler friction.

Run:

```bash
cargo test
```

Each kata exposes a single module `kataXX_*` with one or more `todo!()` functions.

Rules:
- Implement one kata at a time.
- Keep changes minimal.
- After each kata: run `cargo fmt` and `cargo clippy` (or use `./scripts/verify.sh`).
